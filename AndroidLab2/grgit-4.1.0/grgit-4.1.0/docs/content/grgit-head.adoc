== Name

grgit-head - Gets the current `HEAD` commit.

== Synopsis

[source, groovy]
----
grgit.head()
----

== Description

Returns the link:http://ajoberstar.org/grgit/docs/grgit-core/groovydoc/org/ajoberstar/grgit/Commit.html[Commit] that the current `HEAD` points to.
